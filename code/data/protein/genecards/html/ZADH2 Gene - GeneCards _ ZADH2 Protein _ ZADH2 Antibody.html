Summaries for ZADH2 Gene

GeneCards Summary for ZADH2 Gene

ZADH2 (Zinc Binding Alcohol Dehydrogenase Domain Containing 2) is a Protein Coding gene. GO annotations related to this gene include oxidoreductase activity and transferase activity, transferring acyl groups other than amino-acyl groups. An important paralog of this gene is CRYZ.
